# Sinatra Server
A Sinatra backend that utilises a Tic-tac-toe Gem.

# Installation
- Clone this repository to your target folder
- `npm install`

# Running
In the console, run `shotgun config.ru -p 4567`
Navigate to http://localhost:4567

# Testing
- Cypress: `npx cypress open`
- RSpec: `rspec`
